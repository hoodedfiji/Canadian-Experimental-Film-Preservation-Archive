# Metadata Schema

CEFPA uses Dublin Core as the base metadata framework with additional preservation fields tailored for audiovisual materials.

## Core Fields

- Title
- Director
- Release Date
- Format
- Duration
- Description
- Subject Keywords
- Language
- Country
- Rights Statement
- Collection
- Identifier

## Preservation Fields

- Source of Acquisition
- File Format
- Resolution
- Storage Location
- Preservation Status
- Digitization Status
- Condition Assessment
- Preservation Notes

All entries must include core fields. Preservation fields are strongly recommended.
